# repo-privacy

Caption Grab Chrome 插件的隐私政策页面，通过 GitHub Pages 托管。

## 项目简介

本仓库只包含一个静态页面 `index.html`，作为 [Caption Grab](https://github.com/jackdaidai/caption-grab) Chrome 插件的公开隐私政策（Chrome 网上应用店上架所需内容）。页面说明 Caption Grab 不会将任何用户数据收集、传输或存储到外部服务器——所有字幕提取、缓冲与导出都在浏览器本地完成。

## 技术栈

- 纯静态 HTML（无需构建、无依赖）
- 使用 GitHub Pages 托管

## 快速开始

直接在浏览器中打开 `index.html`，或用任意静态文件服务器托管本仓库根目录。

## 项目结构

```
index.html   隐私政策页面（内容与样式均内联，单文件）
```

## 线上地址

通过 GitHub Pages 从本仓库托管发布。
