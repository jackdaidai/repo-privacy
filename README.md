# repo-privacy

Privacy policy page for the Caption Grab Chrome extension, hosted via GitHub Pages.

## Overview

This repo contains a single static page, `index.html`, that serves as the public privacy policy for the [Caption Grab](https://github.com/jackdaidai/caption-grab) Chrome extension (required listing content for the Chrome Web Store). It states that Caption Grab does not collect, transmit, or store any user data on external servers — all caption extraction, buffering, and export happen locally in the browser.

## Tech Stack

- Plain static HTML (no build step, no dependencies)
- Hosted with GitHub Pages

## Quick Start

Open `index.html` directly in a browser, or serve the repo root with any static file server.

## Project Structure

```
index.html   The privacy policy page (self-contained, inline CSS)
```
